# GreenGlobe Logistics - Next.js Homepage

## Setup

```bash
npm install
npm run dev
```

## Stack
- Next.js 14 (App Router)
- Bootstrap 5
- Framer Motion
- React CountUp

## Dynamic areas
- Header logo + nav links in `app/page.js`
- Hero heading, subtext, buttons, background image, and stats in `heroData`
- Services, Why GreenGlobe items, and partners are all data-driven arrays

- Replace placeholder logo, portrait, and hero background URLs with final assets
